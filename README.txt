Bitcointip reddit bot

This bot allows users to control bitcoins with their reddit account.  Users can tip other users or bitcoin addresses within a reddit comment or message.  The bot scans user comments and messages for tips of the form "+bitcointip @Username 1 BTC" and then transfers control of that amount of bitcoins from sender to receiver to withdraw or tip at any time.

Bitcointip Subreddit: http://www.reddit.com/r/bitcointip

State of the bot:  /src/bitcointip.py is in the process of being converted from PHP to python and implementing the Python Reddit API Wrapper (praw)
